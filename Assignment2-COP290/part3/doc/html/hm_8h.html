Detailed Description

A header file containing implementation of Hashmap compatible with threads and locks.

This file contains a struct and related functions to use a hashmap. It also includes acquire_bucket and release_bucket functions to be used when multiple threads use a hashmap, implementing locks for each list in the hashmap.
